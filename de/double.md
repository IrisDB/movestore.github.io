# Zahleneingabe
Um dem Benutzer die Möglichkeit zu geben eine Zahl (Gleitkommazahl) eingeben zu können, kann der Typ `DOUBLE` verwendet werden. 

## Beispiel

```json
{
  "id": "testDouble",
  "name": "Test floating-point number",
  "description": "Select a number.",
  "type": "DOUBLE",
  "defaultValue": null
}
```
